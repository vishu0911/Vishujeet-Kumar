# 🌟 Vishu Portfolio Website

My personal portfolio website showcasing my projects, certificates, and skills, built using HTML, CSS, JavaScript, and optional React.
It’s designed to be responsive, modern, and easy to navigate, highlighting my work for potential employers or collaborators.

# 🚀 Features

💻 About Me section with a short introduction

📂 Projects page showcasing all my completed projects

📜 Certificates page with PDF downloads of achievements

🎨 Responsive and modern UI design

🖼️ Image gallery for certificates and projects

🔄 Easy navigation between sections

# 🛠️ Tech Stack

Frontend: HTML, CSS, JavaScript

Version Control: Git & GitHub

Deployment: GitHub Pages
